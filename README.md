# 🤖 AI FAQ Chatbot

An AI-powered FAQ chatbot built using **Python**, **Streamlit**, **NLTK**, **TF-IDF**, and **Cosine Similarity**. It answers AI-related questions by finding the most relevant response from a predefined FAQ dataset.

## 🚀 Features
- 💬 Chat-based interface
- 🔍 TF-IDF Vectorization
- 📐 Cosine Similarity matching
- 🧠 NLP preprocessing with NLTK
- 💡 Suggested questions
- 🌙 Dark-themed UI
- 🗑️ Clear Chat option

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NLTK
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
